# 로또 번호 추첨기 프로젝트 (단일 파일 버전)

## 개요

이 프로젝트는 사용자가 버튼을 클릭하여 로또 번호를 생성할 수 있는 웹 애플리케이션입니다. 렌더링 문제를 해결하고 단순성을 극대화하기 위해 모든 HTML, CSS, JavaScript 코드를 단일 `index.html` 파일에 포함합니다.

## 디자인 및 기능

### **1. 시각 디자인 (Aesthetics)**

*   **레이아웃:** Flexbox를 사용하여 페이지 내용을 중앙에 배치하고, 로또 번호 볼들을 가로로 정렬합니다.
*   **컨테이너:** 카드 형태의 컨테이너 안에 제목, 로또 번호 영역, 버튼이 포함됩니다.
*   **로또 볼:**
    *   각 로또 번호는 원형의 볼 형태로 표시됩니다.
    *   번호 범위(1-10, 11-20 등)에 따라 다른 배경색이 적용됩니다.
    *   글자색은 항상 흰색이며, 굵은 글씨체로 강조됩니다.
    *   생성 시 애니메이션 효과가 적용됩니다.
*   **버튼:** 그림자 효과와 호버 시 약간의 변형 애니메이션이 있는 녹색 버튼을 사용합니다.

### **2. 핵심 기능**

*   **번호 생성:** '번호 생성하기' 버튼 클릭 시 1부터 45까지의 숫자 중 6개의 고유한 번호를 무작위로 생성합니다.
*   **번호 표시:** 생성된 번호는 오름차순으로 정렬되어 로또 볼 안에 표시됩니다.
*   **페이지 로드 시 자동 생성:** 페이지가 로드될 때 자동으로 로또 번호를 한 번 생성합니다.

### **3. 기술 스택**

*   **HTML:** 단일 `index.html` 파일 내에 모든 구조, 스타일, 로직이 포함됩니다.
*   **CSS:** `<style>` 태그 내에 임베드되어 있으며, 번호대별 색상 클래스와 간단한 애니메이션을 포함합니다.
*   **JavaScript:** `<script>` 태그 내에 임베드되어 있으며, 번호 생성 로직과 DOM 조작을 담당합니다.
